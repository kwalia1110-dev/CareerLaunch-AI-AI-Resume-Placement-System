# CareerLaunch AI – AI Resume Placement System

## Overview
CareerLaunch AI is an AI-powered platform that helps college placement
cells analyze student resumes and identify key skills.

## Features
- Resume upload and analysis
- Skill extraction
- AI-based insights
- Placement support dashboard

## Technologies Used
- Python
- Streamlit
- HTML
- Data Processing

## How to Run
1. Clone the repository
2. Install requirements
3. Run the Streamlit app
